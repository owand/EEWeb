Экспортно-импортная схема





1. ЭКСПОРТЕР по договору комиссии (с комиссией равной таможенным затратам при импорте) на покупку поручает провести импорт ИМПОРТЕРУ.

После осуществления действий указаных в п.2 ЭКСПОРТЕР производит расчеты с таможней налоговым векселем и зачитывается с налоговой, растаможив товар, ЭКСПОРТЕР продает его ИМПОРТЕРУ по цене 100 + НДС 20 + таможенные затраты.

ИМПОРТЕР рассчитывается с ЭКСПОРТЕРОМ векселями V1 (100)+V2 (20) + V3 (там. затраты).

ЭКСПОРТЕР в погашение своего долга по договору комиссии передает вексель V1 (100) + V3 (комиссия равная таможенным затратам).

2. ИМПОРТЕР по договору купли-продажи с НЕРЕЗИДЕНТОМ импортирует товар за счет своих средств по цене 100 и на таможне указывает, что товар принадлежит ЭКСПОРТЕРУ.

3. ЭКСПОРТЕР продает ИМПОРТЕРУ через БАНК вексель V2 (20) с дисконтом.
